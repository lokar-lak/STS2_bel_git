# STS2_bel_git
Поўны пераклад Slay the Spire 2 на беларускую мову ад каманды lokar.

1. Распакаваць [архіў](https://www.nexusmods.com/slaythespire2/mods/1162?tab=files&file_id=5360) у папку **mods** у каранёвай папцы гульні.
   Калі папкі **mods** няма — яе трэба стварыць.
   Звычайна шлях да яе будзе такім:
   - **Windows:** `C:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2\mods`
   - **Linux:** `~/.steam/steam/steamapps/common/Slay the Spire 2/mods`

2. Запусціць гульню, зайсці ў налады моў (Settings → Languages) і выбраць беларускую.

3. Прыемнай гульні!

**P.S.** Калі вы ўсталёўваеце мадыфікацыі, Slay the Spire 2 стварае асобны набор профіляў, каб не пашкодзіць вашыя захаванні. Калі вы хочаце захаваць прагрэс, гуляючы з беларусізатарам, можаце:
   - усталяваць мод [UnifiedSavePath](https://www.nexusmods.com/slaythespire2/mods/6)
   - або перанесці профіль уласнаруч.

**Калі гэты спосаб не спрацаваў — паспрабуйце Спосаб №2.**
